AI Research Paper — Public Speech Analysis Using LLMs

This repository contains the research paper “Primena veštačke inteligencije u analizi i obradi govora kroz transkripciju, verifikaciju i evaluaciju izjava”, prepared as part of a group research project and presented at the YUINFO 2025 conference.

The paper covers

Automatic speech transcription and diarization using Whisper

Fact-checking using LLMs (GPT-4o Mini, DeepSeek, SERPAPI, Jina API)

Consistency checking using RAG architecture and COHERE embeddings

JSON structured outputs, text segmentation (chunking), and classification pipelines

Status

The paper is mostly complete — the core system and results are implemented.
The current PDF represents a near-final version of the work, with only minor improvements possible.
